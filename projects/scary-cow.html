Scary Cow


Mid-term project of my Master's Degree in Video Game Development

A game where you have to run and escape the cows that chase after you. You can use your grappling hook to propulse yourself forward and in the air, gaining some speed an run for safety. Made with a home-made engine we built alongside the game. It uses DirectX for rendering and PhysX for the physic simulation.

Player Controller

I created the player controller and the movement of the grappling mechanic. It's important to get the right feeling and avoid a few traps that can appear when manipulating speeds. I also worked on the camera of the player with three views : first person, third person and spectator.

Shadows

I worked on the shadows of the game. We had two types of shadows. First the shadows that each object project on itself. And second the shadow that is projected on other objects. On the image above, the light shadow is cast by the mountains, while the darker shadow is cast by the tunnel.

Ghosts

To provide some sort of a multiplayer experience, I added a Ghost gamemode. It allows you to load one or multiple save files to run against you. No cows in this mode, it is only you and the ghosts you are trying to beat.

Object Editor

To create the map, we have three methods. The first one is to create the terrain based on a heightmap - a grayscale image that represents the height of the terrain. Then we use a random positionner that create random crystals in the tunnel. All of them are placed on the ground or the walls. And finally we have our objects that can be grabbed by the hook as well as checkpoints (invisible objects that activate when they detect a collision with the player). To place those last objects on the map, I created a tool that allows us to place an object at a position and tweak its position, rotation and scale. Those informations are stored in a file. To edit them, we change them in the file and hit a key to reload the objects. We can see our modifications without closing the game! That was quite useful.
